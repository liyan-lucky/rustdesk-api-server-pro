# DeepSeek for Word 使用说明

## 概述

DeepSeek for Word 是一个功能强大的VBA宏，将DeepSeek AI能力集成到Microsoft Word中，提供智能文本处理功能，同时保留原文格式。

## 安装步骤

### 1. 导入宏到Word

1. 打开Microsoft Word
2. 按 `Alt + F11` 打开VBA编辑器
3. 在左侧项目资源管理器中，右键点击您的文档或模板
4. 选择"导入文件"
5. 选择 `DeepSeek-API.bas` 文件
6. 保存文档

### 2. 配置API密钥

首次使用时需要配置DeepSeek API密钥：

1. 从 https://platform.deepseek.com/ 注册账号并获取API密钥
2. 在Word中运行 `DS_Configure` 宏
3. 按照提示输入API密钥

## 主要功能命令

### 文本处理功能

| 命令 | 功能 | 快捷键建议 |
|------|------|------------|
| `DS_Rewrite` | 智能改写选中文本 | Ctrl+Shift+R |
| `DS_Translate` | 翻译选中文本 | Ctrl+Shift+T |
| `DS_Polish` | 润色选中文本 | Ctrl+Shift+P |
| `DS_Summarize` | 总结选中文本 | Ctrl+Shift+S |
| `DS_Expand` | 扩写选中文本 | Ctrl+Shift+E |
| `DS_Custom` | 自定义处理 | Ctrl+Shift+C |

### 配置命令

| 命令 | 功能 |
|------|------|
| `DS_Configure` | 配置API参数 |
| `DS_Help` | 显示使用说明 |

## 使用步骤

### 基本使用

1. **选择文本**：在Word文档中选择要处理的文本
2. **运行宏**：按 `Alt + F8` 打开宏对话框
3. **选择功能**：选择相应的DS_*宏命令
4. **等待处理**：系统将调用DeepSeek API处理文本
5. **查看结果**：处理完成后，文本将被替换为AI生成的内容

### 翻译功能

- 支持多种语言翻译
- 默认目标语言为中文(zh)
- 支持的语言代码：zh(中文)、en(英文)、ja(日文)、fr(法文)、de(德文)、es(西班牙文)等

### 自定义处理

可以输入任何自然语言指令，例如：
- "将文本改为更正式的语气"
- "将内容转换为诗歌风格"
- "用更简单的语言重写"
- "添加更多技术细节"

## 配置选项

### API设置

通过 `DS_Configure` 可以配置：

- **模型选择**：deepseek-chat、deepseek-coder等
- **温度值**：0.0-2.0，控制创造性
- **最大令牌数**：1-4096，控制响应长度

### 默认配置

- API URL: https://api.deepseek.com/v1/chat/completions
- 默认模型: deepseek-chat
- 默认温度: 0.7
- 默认最大令牌数: 2048

## 高级功能

### 格式保留

系统会自动保留原文的格式，包括：
- 字体样式（粗体、斜体、下划线）
- 字体大小和颜色
- 段落对齐和缩进
- 段落间距

### 错误处理

- 网络连接失败时会显示错误信息
- API密钥无效时会提示重新输入
- 处理超时会有相应提示

## 故障排除

### 常见问题

1. **API调用失败**
   - 检查网络连接
   - 验证API密钥是否正确
   - 确认API调用额度是否充足

2. **宏无法运行**
   - 确保已正确导入宏文件
   - 检查Word宏安全性设置
   - 确认文档已启用宏

3. **格式丢失**
   - 确保选择了有效的文本范围
   - 检查Word版本兼容性

### 技术支持

- 确保使用Office 2010及以上版本
- 推荐使用最新版本的Word
- 如遇问题，可运行 `DS_Help` 查看详细说明

## 安全提示

- API密钥请妥善保管，不要分享给他人
- 建议定期更换API密钥
- 处理敏感文档时请注意数据安全
