# Support Policy

This project is provided as open-source software without warranty.

## Community Support

Use GitHub issues for:

- reproducible bugs;
- documentation problems;
- Docker/OpenWrt deployment issues;
- compatibility gaps;
- non-sensitive feature requests.

Please include:

- project version or commit;
- deployment mode;
- operating system and architecture;
- Docker image tag if applicable;
- sanitized logs;
- reproduction steps.

## Do Not Post Publicly

Do not post:

- passwords, password hashes, OAuth client secrets, SMTP credentials, tokens, private keys;
- production `server.yaml` files;
- `server.db` or database dumps;
- uploaded recordings or personal files;
- exploit details for unfixed vulnerabilities.

Sensitive security reports should follow `SECURITY.md`.

## Compatibility Support

This project may include compatibility endpoints or placeholder behavior. Compatibility with any particular RustDesk client, OAuth provider, reverse proxy, OpenWrt version, Docker version, or third-party service is not guaranteed.
